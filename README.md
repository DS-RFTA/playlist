Este projeto nasceu de uma necessidade real: organizar o caos que virou as sugestões de músicas da minha família para a nossa próxima viagem. Ao invés de anotar tudo em um bloco de notas perdido no celular, decidi unir o útil ao agradável, colocando meus estudos em prática e criando nosso próprio gerenciador de playlist no terminal!

## 💡 Sobre o Projeto

O **RoadTrip Playlist Manager** é um sistema interativo desenvolvido em Python (via Prompt/CLI). Ele tem um objetivo simples: cadastrar as músicas que vão tocar no carro e gerar uma lista organizada para a viagem. 

Para garantir que a gente consiga brincar no carro separando as músicas por décadas ou decidir quem vai ser o cantor da vez, o sistema salva as seguintes informações de cada faixa:
*   **Título** da música
*   **Cantor** (ou Banda)
*   **Ano de produção**

## 🛠️ Funcionalidades

*   **Inserção Dinâmica:** Cadastre quantas músicas quiser, de forma sequencial, direto no terminal.
*   **Geração de Playlist:** Visualize todas as músicas cadastradas em uma lista formatada e fácil de ler.
*   **Interface Simples:** Totalmente baseado em texto, focado na praticidade e na lógica de programação.
